Pseudo class

-pseudo-classes are used to add styles to selectors, but only when those selectors meet certain conditions.

A pseudo class is expressed by adding a colon (:) after a selector in CSS.

Here are some commonly used pseudo classes are as shown below:

  • :hover-it is used to select and style element when the mouse pointer is over it.

  • :active-it is used to select and style an element whille it is activated or clicked.

  • :focus-it is used to select and style an element when it receives focus.

  • :visited-it is used to select and style a link that is visited.

  • :link-it is used to select and style a link that is not visited

  • :first-child-it is used to select and style the first child element of its parent.

  • :last-child-it is used to select and style the last child element of its parents.

  • :nth-child(n)-it is used to select and style the nth child element of its parents,we can use different expression for n,like even,odd,numbers,or formulas

  • :empty-it select and style element that has no child or no any content.

  • :checked-it selects and style an input element that is checked.

  • :disabled-it selects and style a disabled input element.

  • :enabled-it selects and style a enabled input element.

  • :required-it selects and style a required input element.

  • :valid-it selects and style an input element with a valid value.

  • :invalid-it selects and style an input element with a invalid value.
