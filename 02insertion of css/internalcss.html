Internal CSS

-it is used for if a single html page has a unique style.

-it is defined inside the <style> element,in head section of html page.

-it is not recomended by coding point of view,because if we have multiple pages in our website,then we have to write CSS in all pages to beautify every page.

Example:

<html>

<head>

<style>

h1 {

color: blue;

margin-left: 20px;

}

</style>

</head>

<body>

<h1>heading</h1>

<p>paragraph</p>

</body>

</html>
