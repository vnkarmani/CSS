Inline css

-this type is used to apply a unique style for a single element.

-to use an inline css,it is used within the the tag with the help of style attribute.

-it is not recomended as per SEO rule,defined by W3C.

-it is applied for single element.

Example:

<body> <h1 style="color:blue;text-align:center;">this is heading</h1> </body>
